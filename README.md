# AI Coding Standards

Open-source **Angular architecture governance** for the [Cursor Marketplace](https://cursor.com/marketplace).

## Public launch

Follow **[PUBLIC_LAUNCH.md](./PUBLIC_LAUNCH.md)** — step-by-step marketplace submission.

```bash
cd angular-standards/mcp-server && npm run build
node ../scripts/validate-marketplace.mjs
```
